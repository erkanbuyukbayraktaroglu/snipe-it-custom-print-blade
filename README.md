# snipe-it-custom-print-blade
Laravel Blade yapısı kullanılarak hazırlanmış bir zimmet formu çıktısı şablonudur. Amaç; bir personele teslim edilen cihaz, aksesuar ve ilgili bileşenlerin kurumsal formatta listelenmesi, teslim ve iade süreçlerinin kayıt altına alınması ve çıktısının yazdırılabilir/PDF’e dönüştürülebilir bir yapıda sunulmasıdır.
